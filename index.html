Vacant House Bingo

A game I put together for the Vacant House Workshop that was intended to teach people about the effects of racial discrimination on housing abandonment. A script and outline for how the game is played is included as part of the workshop script.

A Game about Tax Lien Sales in Baltimore

This is a "choose your own adventure" game created with Twine based on material from a Abell Foundation report on Tax Lien Sales. It is pretty rough but is intended to demonstrate the potential for Twine to teach people about this issue. You can play it here. Here is how you can edit this game:

  1. Download the file: “tax-lien-sale-twine.html”
  2. Download Twine for Windows or OS X. Unzip the downloaded file and then open the application.
  3. Another option is just using the Twine editor online.
  4. Click on “Import From File” (look below the green “+Story” button)
  5. Click on “Choose File” then select “tax-lien-sale-twine.html” to import the game into Twine.
  6. Double-click on any of the passages to edit the text. To create a new passage linked to your current passage just enclose any word or phrase with a pair of double-brackets [[like this]].
